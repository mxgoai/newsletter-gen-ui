## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (version 14.x or higher)
- [npm](https://www.npmjs.com/) (version 6.x or higher) or [yarn](https://yarnpkg.com/) (version 1.22.x or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mxgoai/newsletter-gen-ui.git
   cd newsletter-gen-ui
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Development

To start the development server:

```bash
npm start
# or
yarn start
```

The application will open in your default browser at [http://localhost:3000](http://localhost:3000).

### Building for Production

To create a production build:

```bash
npm run build
# or
yarn build
```

The optimized build will be created in the `build` folder.

### Testing

Run the test suite:

```bash
npm test
# or
yarn test
```
